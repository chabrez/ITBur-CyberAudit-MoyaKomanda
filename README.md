## Описание проекта

Консольное Python-приложение выполняет автоматизированный аудит безопасности Linux-системы, обнаруживает типовые ошибки конфигурации и находит флаги CTF формата `bit26{...}`.

### Основные возможности:
- **Анализ прав доступа**: поиск файлов с опасными правами (777, 666) в директориях /etc, /var, /home
- **Сетевой аудит**: обнаружение открытых опасных портов (FTP, Telnet, MySQL)
- **Аудит пакетов**: проверка версий критического ПО
- **Поиск флагов**: автоматическое обнаружение флагов `bit26{...}` в файлах
- **Рекомендации**: конкретные команды для исправления каждой проблемы

## Требования к системе

- **ОС**: Linux (Kali Linux, Ubuntu, Debian)
- **Python**: версия 3.6 или выше
- **Права**: рекомендуется запуск с `sudo` для полного доступа к файлам
